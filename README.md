# Student Dashboard

A responsive Student DashBoard made using React.
This project allows users to view student details, navigate between pages, and add new students dynamically using a form.


## Features

- **Home Page:** Display all student card including Name, Email, and City.
- **Student Details Page:** Displays detailed profile information when a card is clicked.
- **Add Student Page:** Includes a form with input validation, stores student data locally.
- **Local Storage:** Ensures local data persists after page refresh.
- **Navigation Bar:** Easily Navigate between HomePage, and Add Student Page.


## Tech Stack

- React JS
- React Rounter v6
- Local Storage API
- CSS3

## Installation and Setup

1. Clone the repository

```
git clone https://github.com/HimanshuTech0509/benchmarking-assignment-himanshu.git
cd student-dashboard

```

2. Install dependencies

```
npm install
```

3. Start the development server

```
npm run dev
```

4. Open the app in your browser at:

```
http://localhost:5173/

```
